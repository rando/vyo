== Description ==
Add template files through admin screen, And view listing for all custom template files in theme with linked pages.
Easy to use and helpful for theme developers to check template connection to pages and help find out the exact file.

== Installation ==

This section describes how to install the plugin and get it working.

In WordPress:

    Go to Plugins > Add New > search for :: Add custom page template ::
    Press Install Now for the Show notice or message on admin area plugin
    Press Activate Plugin

To install manually instead:

    Upload the Add-custom-page-template  directory to the /wp-content/plugins/ directory
    Activate the plugin through the Plugins menu in WordPress
